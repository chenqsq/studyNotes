案例22：Flex项目伸缩比例

使用flex-grow和flex-shrink属性控制Flex项目的伸缩比例

核心属性：flex-grow和flex-shrink

flex-grow (扩展比例)

grow:1
grow:2
grow:1

flex-shrink (收缩比例)

shrink:1
shrink:2
shrink:3
/* 核心Flex代码 */
/* flex-grow示例 */
.item-1 { flex-grow: 1; }
.item-2 { flex-grow: 2; }
.item-3 { flex-grow: 1; }

/* flex-shrink示例 */
.item-1 { flex-shrink: 1; }
.item-2 { flex-shrink: 2; }
.item-3 { flex-shrink: 3; }
